# Pre-v3 archived runs

Empirical runs from the early DSPy ReAct + manual experience library era. Kept for reproducibility; not used to compute v3 headline numbers.

Subdirs:
- `musique/`, `2wikimultihop/`, `hotpotqa/`, `bamboogle/` — 1000q-scale ReAct / GPT-4o-mini-sub baselines
- `musique_stratified100/` — strat100 prompt-tuning sweeps
- `smoke/` — small smoke tests
- `diagnostics/` — pre-v3 diagnostic JSONs (not the same as `results/diagnostics/`)

The current v3 results live in `results/runs/test_v3_cand13/` and ablation runs that follow it.
